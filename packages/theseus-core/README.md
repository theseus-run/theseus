# @theseus.run/core

Typed primitives for Theseus agent runtimes.

```ts
import * as Tool from "@theseus.run/core/Tool";
import * as Dispatch from "@theseus.run/core/Dispatch";
import * as Capsule from "@theseus.run/core/Capsule";
```

Core exports the Tool primitive, dispatch loop, agent result types, Capsule
service, satellite middleware, and bridges to `effect/unstable/ai`.

This package targets Bun and exports TypeScript source.
